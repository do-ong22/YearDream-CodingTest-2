판매액 현황 보고서 작성하기 
주문 데이터`sales.csv` 는 아래와 같은 정보를 담고 있습니다.

| 열 이름        | 열 설명         |
|---------------|----------------|
| customer      | 주문자          |
| order_sales   | 주문별 판매액   |
| order_month   | 주문 월         |
| order_type    | 주문 형태       |
| order_satis   | 주문 만족도     |

주문 데이터에서 **판매액**에 대한 평균, 중간값, 최빈값, 표준편차, 백분위수를 파악하는 코드를 지시사항에 맞춰 작성하세요.

## 지시사항
`sales.csv` 를 불러와 저장한 데이터프레임 `sales`를 이용해 주문 데이터의 주문별 판매액 ("order_sales")에 대한 아래 정보들을 저장하세요.

### 1. 판매액 평균 확인
전체 판매액의 평균을 계산하여 변수 `sales_mean` 에 저장하세요. 소숫점 첫째자리에서 반올림하여 **정수 부분**만 저장하세요.

### 2. 판매액 중앙값 확인
전체 판매액의 중앙값을 계산하여 변수 `sales_median` 에 저장하세요. 

### 3. 판매액 최빈값 확인
전체 판매액의 최빈값을 계산하여 변수 `sales_mode` 에 저장하세요.

### 4. 판매액 표준편차 확인
전체 판매액의 표준편차를 계산하여 변수 `sales_std` 에 저장하세요. 소숫점 첫째자리에서 반올림하여 **정수 부분**만 저장하세요.

### 5. 판매액 30,70 분위수 확인
전체 판매액의 30, 70 분위수를 계산하여 각각 변수 `q30`, `q70` 에 저장하세요.

### 주의사항

- 지시사항에서 명시한 부분 외에 스켈레톤 코드를 임의로 삭제하거나 수정할 경우, 혹은 명시된 변수명 또는 함수명을 사용하지 않을 경우 정상적으로 채점이 되지 않을 수 있습니다. 반드시 지시사항을 확인해주세요.

- 우측 상단의 코드 초기화 버튼을 통해 코드를 초기 상태로 되돌릴 수 있습니다.
